# Linka
[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy)  
My Slack bot for notifying summary of result in Groove Coaster 3.  

===

## Description  
When send `linka` to Linka, Linka returns result.
